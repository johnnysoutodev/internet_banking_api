# Internet Banking API

API referente ao teste da oportunidade de Desenvolvedor Back-End Pleno no Santander.

Neste projeto foi utilizadas as seguintes tecnologias:

- Spring Boot
- Spring Doc
- Swagger UI
- Maven
- Database H2
- IDE IntelliJ IDEA Community Edition
- Java JDK 17

Para utilização deste projeto baixe este projeto, salve em um local do seu computador de sua preferência, apos abra o projeto pela IDE recomendada, IntelliJ IDEA Community Edition,faça a sincronização das dependências do Maven, apos concluir o download das dependências, abra o arquivo InternetBankingApiApplication e clique em Run.

Apos a aplicação estiver rodando acesse o link:

[http://localhost:8080/api/docs/index.html](http://localhost:8080/api/docs/index.html)

Neste link você terá acesso a documentação, neste documentação esta sendo entregue três métodos.
